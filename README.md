# aJtrace

trace java function invoke use aspectJ

# 依赖

- web服务器与php
- php要对`output`文件夹有读写的权限

# 使用

- 需要安装`aspectj`，且用安装目录下的`aspectjrt`与`ajc`替换该项目`lib`目录下的同名文件。
- 目前`lib`目录下的文件是我电脑上的版本，不具备通用性，可查看`lib/ajc`（是文本文件）文件了解。
